CodeMaster is a full-stack coding practice platform that allows users to solve programming problems, track their progress, and explore solutions through a modern, interactive UI. The platform supports code submissions via an external judge, user authentication, progress analytics, and AI-assisted tools to help users understand solutions and complexity.

## Tech Stack

- **Backend:** Node.js, Express.js, MongoDB (Mongoose), Redis, JWT, bcrypt
- **Authentication:** Email/Password, Google OAuth, JWT
- **AI & Services:** AI helpers for explanations and complexity analysis, Cloudinary for media uploads, Nodemailer for emails
- **Frontend:** React.js, Vite, Redux Toolkit, React Router, Tailwind CSS, DaisyUI
- **Code Execution:** Judge0 integration for running and evaluating code

## Features

- **Problem Solving:** Browse, filter, and solve coding problems with real-time code submission and evaluation using Judge0, supporting multiple programming languages.
- **Multi-Language Support:** Time-bound coding environment with support for different programming languages and standardized execution constraints.
- **Personalized Notes:** Add and manage personal notes for each problem to record approaches, mistakes, and key learnings.
- **Video Courses:** Structured, topic-wise video courses to learn data structures, algorithms, and problem-solving techniques.
- **Video Solutions:** Dedicated video explanations and walkthroughs for individual coding problems.
- **Contests:** Timed coding contests with rankings, submissions, and performance tracking.
- **Timer-Based Environment:** Built-in timer for practice and contest modes to simulate real interview and competitive coding environments.
- **Authentication:** Secure user authentication with email/password, Google OAuth, and email verification.
- **Progress Tracking:** Track solved, attempted, and bookmarked problems with activity graphs and leaderboards.
- **AI Assistance:** AI-powered explanations and time/space complexity analysis for submitted solutions.
- **Profile Management:** User profiles with statistics, activity history, and avatar uploads.


Admin Tools: Admin panel for managing users, problems, contests, and platform data.

Theme Support: Dark and light mode toggle for an improved and accessible user experience.
